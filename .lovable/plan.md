

# คู่มือการใช้งาน — Rich Menu ปุ่มที่ 6

## วิธีตั้งค่าใน LINE Official Account Manager

ในช่อง **Action** ของปุ่มที่ 6 เลือก **Text** แล้วใส่ข้อความนี้ (15 ตัวอักษร พอดี ๆ):

```
ดูคู่มือการใช้งาน
```

เมื่อ user กดปุ่ม → LINE จะส่งข้อความ "ดูคู่มือการใช้งาน" เข้า chat → bot ตรวจจับและตอบ **Flex Message คู่มือฉบับเต็ม** กลับ (ไม่จำกัด 50 ตัวอักษร)

## สิ่งที่ฉันจะทำในโค้ด

แก้ไฟล์ `supabase/functions/line-webhook/index.ts` (บรรทัด ~365):

1. **เพิ่ม trigger** `ดูคู่มือ`, `คู่มือ`, `manual`, `guide` ใน regex help command เดิม
2. **สร้างฟังก์ชัน `getUserGuideFlex()`** — Flex Bubble แบ่งเป็นส่วน:
   - **Header**: "📖 คู่มือการใช้งาน Cruzee Finance"
   - **ทีมงาน (Staff)**: วิธีลงทะเบียน, ส่งใบวางบิล, ผูก LINE, เช็คสถานะการจ่ายเงิน
   - **คู่ค้า (Vendor)**: วิธีลงทะเบียนคู่ค้า, ส่งใบแจ้งหนี้/บิล, อัปโหลด ภพ.20
   - **คำสั่ง LINE**: รายการคำสั่งพิมพ์ได้ (เช่น "วางบิล", "สถานะ", "help")
   - **Footer button**: เปิด Portal ผ่าน LIFF URL `https://liff.line.me/2008893199-xaJITz5y?view=quick-link&owner=...`
3. **ตอบกลับด้วย `replyFlexToUser()`** (มีอยู่แล้วในไฟล์)

## คำสั่งที่ bot จะรู้จัก (รวมของเดิม)

`ดูคู่มือการใช้งาน`, `ดูคู่มือ`, `คู่มือ`, `help`, `วิธีใช้`, `menu`, `เมนู`, `?`
