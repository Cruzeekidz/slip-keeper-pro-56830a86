

# แผนเพิ่มลายเซ็นและตราประทับบนหนังสือรับรองหัก ณ ที่จ่าย

## สิ่งที่จะทำ

1. **เก็บรูปลายเซ็น+ตราประทับเข้าโปรเจค** — Copy รูปที่อัปโหลดไปไว้ที่ `src/assets/company-stamp.png` เพื่อใช้ในการ Generate PDF

2. **แปลงรูปเป็น Base64 ฝังใน PDF** — เนื่องจาก PDF สร้างด้วย `window.open()` + HTML ต้องแปลงรูปเป็น base64 data URL เพื่อให้แสดงผลได้โดยไม่ต้องโหลดจาก server

3. **เพิ่มลายเซ็น+ตราประทับในส่วนลงนาม** — ในฝั่ง "ผู้จ่ายเงิน" ของ PDF จะแสดงรูปลายเซ็นและตราประทับอัตโนมัติ พร้อมชื่อ "บริษัท เม้งซินเทรดดิ้ง จำกัด" ส่วนฝั่ง "ผู้รับเงิน" ยังคงเว้นว่างให้เซ็นจริง

## ไฟล์ที่แก้ไข

| ไฟล์ | รายละเอียด |
|------|-----------|
| `src/assets/company-stamp.png` | รูปลายเซ็น+ตราประทับบริษัท |
| `src/pages/WhtCertificateForm.tsx` | แก้ฟังก์ชัน `generatePDF` — import รูปเป็น base64 แล้วแทรกใน sign-section ฝั่งผู้จ่ายเงิน |

## รายละเอียดทางเทคนิค

- Import รูปผ่าน Vite: `import companyStampUrl from "@/assets/company-stamp.png"`
- ใช้ `fetch()` + `FileReader` แปลงเป็น base64 data URL ก่อนสร้าง HTML
- ในส่วน sign-section ของ PDF HTML:
  - ฝั่งซ้าย (ผู้จ่ายเงิน): แสดงรูปลายเซ็น+ตราประทับ ขนาดประมาณ 180px พร้อมชื่อบริษัทด้านล่าง
  - ฝั่งขวา (ผู้รับเงิน): เว้นว่างเหมือนเดิม ให้ผู้รับเงินเซ็นจริง
- เพิ่มวันที่ออกเอกสารอัตโนมัติใต้ลายเซ็นผู้จ่ายเงิน
